collapsed

Indicates if the element is collapsed or expanded.

Description

If collapsed is true, the element is not displayed on the page under any circumstances. A collapsed element, unlike a hidden element, does not take up any space on the page. When collapsed, elements positioned within 70 pixels below the collapsed element and each other will move up to take the collapsed element's place where possible. The elements that move up maintain their positions relative to one another.

If collapsed is false, the element may be displayed on the page. Elements that moved up to take the collapsed element's place on the page are moved back down.

However, an expanded element (an element whose collapsed property is false) will still not be displayed if:

  • It is hidden.
  • Its parent element is hidden or collapsed.
  • It is a slide in a Slideshow which is currently not being displayed.

Even if the element will not be displayed due to the conditions mentioned above, if its collapsed property is false, it will be displayed when the conditions no longer apply.

To set the collapsed property on an element, use the element's collapse() and expand() functions.

If you select Collapsed on load in the element's Properties panel in the Editor, the collapsed property is set to true when the page loads.

Syntax

get collapsed(): boolean

TYPE

Boolean

DEFAULT VALUE

false

Examples

Get the collapsed status of an element

let isCollapsed = $w("#myElement").collapsed; // false

Toggle the collapsed state of an element

if( $w("#myElement").collapsed ) {
    $w("#myElement").expand();
}
else {
    $w("#myElement").collapse();
}

show()

Shows the element and sets its hidden property to false, using an animation if specified.

Description

The show() function returns a Promise that is resolved when the animation is complete and the element's hidden property has been set to false.

To learn about the behavior of a shown element, see the hidden property.

You can optionally apply one of the following animations when showing the element:

  • "ArcIn"
  • "BounceIn"
  • "DropIn"
  • "ExpandIn"
  • "FadeIn"
  • "FlipIn"
  • "FloatIn"
  • "FlyIn"
  • "FoldIn"
  • "GlideIn"
  • "Reveal"
  • "SlideIn"
  • "SpinIn"
  • "TurnIn"

See Also

hide(), hidden, expand()

Syntax

function show([animationName: ShowAnimation]): Promise<void>
type ShowAnimation = "ArcIn" | "BounceIn" | "CollapseIn" |
  "FadeIn" | "FlipIn" | "FloatIn" | "FlyIn" | "FoldIn" | "GlideIn" |
  "PopIn" | "Reveal" | "SlideIn" | "SpinIn" | "TurnIn"

PARAMETERS

animationName
(Optional)
 string
The name of the animation to trigger when showing the item.

RETURNS

Promise<void> When the animation is complete and the element's hidden property has been set to false.

MIXED IN FROM

$w.HiddenCollapsedMixin

Examples

Show an element with no animation

$w("#myElement").show();

Show an element with the "FadeIn" animation

$w("#myElement").show("FadeIn");

Show an element with an animation and log message when the animation is done

$w("#myElement").show("FadeIn")
  .then( ( ) => {
    console.log("Done with fade");
  } );

Toggle the hidden state of an element

if( $w("#myElement").hidden ) {
  $w("#myElement").show();
}
else {
  $w("#myElement").hide();
}
